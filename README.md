# Thejus Pradeep - Personal Portfolio Website

A premium, modern, and pixel-perfect personal portfolio website designed exactly to match the high-end aesthetic layout of the approved reference design. This single-page application is tailored for a software developer/creative engineer, highlighting academic milestones, technical competencies, and elegant interactive capabilities.

---

## 🌟 Key Features

- **High-End Aesthetics**: Built using rich warm gradient tones, sleek dark background structures (`#0B0B0C`), custom glowing lighting indicators, and high contrast typography.
- **Glassmorphic Navigation**: Sticky top navigation bar featuring background blur backdrop-filters, custom sliding active-section highlighting, and interactive pill-buttons.
- **Vibrant Hero Card**: Features a custom large curved card container at the base (`border-radius: 40px` to `110px` responsively) layering a professional silhouette against a fiery-orange studio backlight gradient.
- **Client & Competency Marquee**: Monochrome tech-icon layouts highlighting core stack frameworks (React JS, Node, Python, Git).
- **"Behind the Systems" (About)**: Two-column minimalist representation introducing the engineering focus and study tracks.
- **Interactive Skills Radar**: Visual percentage-based proficiency cards grouping Core Languages, Libraries/Frameworks, and Dev-Operations.
- **Grayscale Case Study Cards**: Mirroring the design cards structure, these items use high-contrast monochrome images that scale (`scale-104`) and apply colorful visual blend masks on hover triggers.
- **Academic Timeline**: Clean, minimal vertical chronological tracker highlighting academic context at **Vimal Jyothi Engineering College, Chemperi** in Kannur, Kerala.
- **Premium Contact Drawer**: Interactive float-up input labels, animated focus rings, custom submit handlers with button state loading triggers, and a successful message validation toast.
- **Responsive Layout**: Designed and tested for fluid styling across Ultra-Wide, Desktop, Tablet, and Mobile Viewports.

---

## 🛠️ Tech Stack & Technologies

1. **HTML5**: Semantic tags, accessibility attributes, and SEO optimization.
2. **Tailwind CSS v3 (via CDN)**: Utility-first, robust grids, flex-containers, and visual spacing tokens.
3. **Custom Vanilla CSS** (`styles.css`): Dynamic curved elements, background lighting blur elements, floating form labels, card hue-tint overlays, and transition animation curves.
4. **Vanilla JavaScript** (`app.js`): Sticky headers, mobile drawer toggles, active link tracking (Intersection Observer API), viewport scroll-reveal loaders, card touch event overrides, and async contact form animations.

## 🚀 Local Setup & Preview

Since the application is built using lightweight high-performance static architectures, you do not need complex installation configurations or compilers:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/thejus06/portfolio-v1.git
   cd portfolio-v1
   ```

2. **Spin Up a Local Development Server**:
   You can preview your website live locally by launching a standard Python server:
   ```bash
   python -m http.server 8000
   ```

3. **Browse the Website**:
   Open your browser to: **[http://localhost:8000](http://localhost:8000)**
